📋プレイリストについて

    プレイリストの作成・編集

    1. ホーム画面のハンバーガーメニューmenuを表示し、プレイリストを押下してください。

      2. ホーム画面のqueue_musicボタンを押下して表示されるプレイリスト選択ダイアログの下にあるプレイリストを作成ボタンを押下することでも、プレイリスト一覧画面に移動できます。

    2. プレイリスト一覧画面の新規作成ボタンを押下して、プレイリスト詳細作成画面に移動してください。

    1曲ずつ選択する場合

    1. プレイリスト詳細画面のfile_openボタンを押下して、ファイル選択ダイアログを表示してください。
    2. プレイリストに登録したい曲を押下してください。

    フォルダを丸ごと選択する場合

    1. プレイリスト詳細画面のfolderボタンを押下して、フォルダ選択ダイアログを表示してください。
    2. プレイリストに登録したいフォルダを押下して、フォルダ選択ダイアログ下部のcheck_circle
      3. を押下すると、フォルダ内の曲がすべてプレイリストに登録されます。
      また、同時に選択したフォルダ名がプレイリスト名に設定されます。

      「サブフォルダ」以下の音楽ファイルはプレイリスト登録対象になりません。

    曲の並び替え・削除

    1. プレイリストに登録した曲には、以下の操作を行うことができます。
      2. keyboard_arrow_up 再生順をひとつ前に移動させる
      keyboard_arrow_down 再生順をひとつ後ろに移動させる
      delete 曲をプレイリストから削除する
    2. プレイリスト詳細画面上部のプレイリスト名入力欄でプレイリストの名前を入力すると、保存ボタンが押下できるようになります。
    3. 保存ボタンを押下すると、保存完了ダイアログが表示されます。
    4. 保存完了ダイアログのOKボタンを押下すると、プレイリスト詳細画面からプレイリスト一覧画面に自動的に戻ります。
    5. プレイリスト横のボタンで、以下の操作を行うことができます。
      6. edit プレイリストを編集
      delete プレイリストを削除

    プレイリストの再生

    1. ホーム画面のqueue_musicボタンを押下して、プレイリスト選択ダイアログを表示してください。
    2. 再生したいプレイリスト名を押下すると、再生モードボタンのアイコンの状態に応じた順番でプレイリストに登録した曲を再生します。
      3. restart_alt 登録順にループ再生
      shuffle シャッフル再生

      プレイリストに登録された曲がすべて削除/移動/リネームされている場合、再生に失敗します(一部の曲が残っている場合、見つかった曲だけを再生します)。その場合は曲の登録をやり直してください。

    プレイリストの削除

    1つだけ削除する場合

    1. プレイリスト一覧画面で、削除したいプレイリストのdeleteを押下してください。
    2. 確認ダイアログでOKを押下してください。
    3. 削除が完了すると、トーストメッセージが表示されます。

    すべてのプレイリストを削除する場合

    1. プレイリスト一覧画面で、全てのプレイリストを削除を押下してください。
    2. 確認ダイアログでOKを押下してください。
    3. 削除が完了すると、トーストメッセージが表示されます。
