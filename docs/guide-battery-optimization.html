🔋バッテリー使用量の最適化

お使いのスマートフォンによってボタンの名称・手順が異なる場合があります。

    Google端末(ピュアandroid)の場合

    1. 設定のアプリを押下してください。
    2. ○○個のアプリをすべて表示を押下してください。
    3. 表示されるアプリ一覧の中からBaconJamを探して押下し、アプリのバッテリー使用量を押下してください。
    4. バックグラウンドでの使用を許可をオンにしてください。

    OPPO端末(ColorOS)の場合

    1. 設定のアプリを押下してください。
    2. アプリ管理を押下してください。
    3. 表示されるアプリ一覧の中からBaconJamを探して押下し、バッテリー使用量を押下してください。
    4. バックグランドアクティビティを許可をオンにして、表示されるダイアログの許可を押下してください。

    Nothing端末(Nothing OS)の場合

    1. スマートフォンの設定→アプリを押下してください。
    2. ○○個のアプリをすべて表示を押下してください。
    3. 表示されるアプリ一覧の中からBaconJamを探して押下し、アプリのバッテリー使用量を押下してください。
    4. アプリのバッテリー使用量画面に表示されている
      • 制限なし
      • 最適化
      • 制限
      の中から、制限なしを選択してください。
