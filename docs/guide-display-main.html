ホーム画面の操作

画面要素 機能
send Spotify、YouTube Musicなどで再生中の曲を強制的になうぷれ投稿します。
でなうぷれ投稿するを無効にしている場合でも投稿するので、特定の曲のみ投稿したい場合に使用してください。
translate アプリの表示言語を切り替えます。
詳細な操作方法はこちらのページをご覧ください。
music_note 再生中の音楽ファイルの曲名を表示します。
曲名が画面に収まらない場合はスクロール表示します。
再生中の音楽ファイルに埋め込まれているアルバムアートワークを表示します。
埋め込まれていない場合は左の画像を表示します。
person 再生中の音楽ファイルのアーティスト名を表示します。
アーティスト名が画面に収まらない場合はスクロール表示します。
album 再生中の音楽ファイルのアルバム名を表示します。
アルバム名が画面に収まらない場合はスクロール表示します。
queue_music 再生するプレイリストを選択します。
プレイリストの作成・編集方法はこちらのページをご覧ください。
restart_alt/shuffle プレイリストの再生順序を決定します。
  • restart_alt：登録順にループ再生
  • shuffle：シャッフル再生
music_note 再生する音楽ファイルを選択します。
♭1 再生中の音楽を半音下げます。
0 キーの状態を表示します。
押すとデフォルトのキーに戻すことができます。
＃1 再生中の音楽を半音上げます。
skip_previous 1曲前に戻ります。
play_arrow/pause 曲を再生/停止します。
skip_next 1曲先に進みます。
