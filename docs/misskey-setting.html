との連携

    連携(アクセストークンの発行)

    「なうぷれ投稿」したいMisskeyサーバーで二要素認証の設定が必要な場合は、こちらを参考にあらかじめ設定を行ってください。

    1. 設定画面のMisskey接続設定を押下して、設定項目を展開してください。

      2. Misskey接続設定

    2. 認証するを押下して、アプリへのアクセス許可ページにアクセスしてください。
    3. 「なうぷれ投稿」したいMisskeyアカウントのユーザー名、パスワードを入力してログインを押下してください。
    4. 以下の図のように二要素認証の入力を要求された場合は、こちらの手順で作成した認証コードを入力してログインを押下してください。
    5. 以下の図のようなアクセス許可ページが表示されるので、許可を押下してください。
    6. 以下の図のような認証完了ページが表示されたら、BaconJamに戻ってください。
    7. 連携に成功している場合以下の図のようなダイアログが表示されます。

      8. 許可を押下する前にBaconJamに戻ったり、キャンセルを押下すると認証に失敗します。その場合は、再度認証するを押下するところから行ってください。

    発行したアクセストークンの確認・削除

    1. 連携先のMisskeyサーバーで設定→APIを押下してください。
    2. アクセストークンの管理を押下してください。
    3. アクセストークン一覧に、BaconJam 機種名 発行年_月_日 時_分_秒という名前のものが発行されています。不要になった場合は、ゴミ箱アイコンを押下してアクセストークンを削除してください。
