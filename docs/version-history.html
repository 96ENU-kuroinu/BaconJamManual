1.1.0(バージョンコード3)

リリース日：2024年04月07日

  • 「なうぷれ投稿」の公開範囲を以下から選択できる機能を追加しました。
    1. パブリック(すべてのユーザーに公開)
    2. ホーム(ホームタイムラインのみに公開)
    3. フォロワー(自分のフォロワーのみに公開)
    4. チャンネル
  • Spotifyとの連携機能(プレビュー版)を追加しました。
  • 設定画面に設定をリセットするボタンを追加しました。
  • 「なうぷれ投稿」するアーティスト名を音楽ファイルのメタデータから以下の優先順位で取得し、最初に見つかったものを使用するように変更しました。
    1. アルバムアーティスト名
    2. トラックアーティスト名
    3. 著者名
    4. 作者名
  • ランキング画面の不要なTESTボタンを削除しました。
  • 初回インストール時に投稿パターンの初期値が設定されておらず、「なうぷれ投稿」できない不具合を修正しました。
