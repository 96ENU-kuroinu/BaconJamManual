なうぷれ投稿について

  1. ホーム画面のハンバーガーメニューmenuを押下してドロワーを表示し、設定を押下してください。

    有効/無効化

    • なうぷれ投稿するをオンにすると、音楽再生時になうぷれ投稿されるようになります。
    • Misskeyへの投稿に必要なアクセストークンが発行されていない場合、以下のようなダイアログが表示されます。
    • OKを押下すると、Misskeyとの連携に進みます。

    投稿範囲

    1. 設定画面の「なうぷれ設定」の詳細を押下して、詳細設定を展開して下さい。

      2. なうぷれ詳細設定

    2. 公開範囲のドロップダウンで、以下のいずれかからなうぷれ投稿を選択できます。
      3. パブリック すべてのユーザーに公開
      ホーム ホームタイムラインのみに公開
      フォロワー 自分のフォロワーのみに公開
      チャンネル 特定のチャンネルに公開

      ドロップダウン一覧

    3. ドロップダウンでチャンネルを選択すると、ドロップダウンの下にチャンネルを選択ボタンが表示されます。
    4. チャンネルを選択ボタンを押下すると、チャンネル選択画面に移動します。
    5. チャンネル名またはチャンネルIDを検索欄に入力すると、候補が表示されます。
    6. 表示されたチャンネル名を押下すると、投稿先として選択できます。

    アルバムアートワークの添付

    1. 設定画面の「なうぷれ設定」の詳細を押下して、詳細設定を展開して下さい。

      2. なうぷれ詳細設定

    2. アルバムアートのドロップダウンで、以下のいずれかから添付モードを選択できます。
      • 添付しない
      • Wi-Fi接続時のみ添付する
      • モバイル通信でも添付する

      ドロップダウン一覧
